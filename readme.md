# 📈 Stocks API

A simple RESTful API built with **Golang**, **Gorilla Mux**, and **Postgress** that performs CRUD (Create, Read, Update, Delete) operations on stock records.

## 🚀 Features

- Create a stock
- Get a stock by ID
- Get all stocks
- Update a stock
- Delete a stock
- Postgresql database integration

---

## 🛠️ Tech Stack

- Go (Golang)
- Gorilla Mux
- Postgresql
